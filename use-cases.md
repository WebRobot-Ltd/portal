## LLM Fine-tuning Datasets

Generate high-quality datasets for training and fine-tuning Large Language Models.

**Features:**
- Extract text from public domain sources
- Clean and normalize data
- Generate training-ready datasets
- Support for multiple formats (JSONL, Parquet, CSV)

**Example Use Cases:**
- Training domain-specific models
- Fine-tuning for specific tasks
- Dataset augmentation

[Learn More →](https://docs.webrobot.eu/guides/vertical-llm-finetuning)
